    Circuit:
    
      +5V-----------------,---,----------------+5V
                          |   |
                          |   |    
                        [R3] [R4]    
                          |   |    
                          |   |   C2          Phono
       	    A1]-ADC1------(---+---||------------o R
            A0]-ADC0------+---(---||------------o L       
             		      |   |   C1    ,-------o GND     
                   	      |   |         |
                        [R5] [R6]       |
                          |   |         |
                          |   |         |
      GND-----------------'---'---------'------GND

      ,------------------------,
      |   n   |   R    |   C   |
      |-------+--------+-------|
      |   1   | 10kOhm |  1uF  |
      |   2   | 10kOhm |  1uF  |
      |   3   | 10kOhm |       |
      |   4   | 10kOhm |       |
      '------------------------'

	The input impedance is around 5k.
	The filter corner frequency is around 20Hz.
